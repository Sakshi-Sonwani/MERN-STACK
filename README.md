# MERN Stack CRUD Application

A full-stack CRUD application built using:

- MongoDB
- Express.js
- React.js
- Node.js

## Features

- Create User
- View Users
- Update User
- Delete User

## Project Structure

/client  → React frontend  
/server  → Express backend  

## How to Run

### Backend
cd server  
npm install  
npm start  

### Frontend
cd client  
npm install  
npm start
